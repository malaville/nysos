KeyBoard Shortcuts

  • CTRL + G
    Open groupping tool 
  • CTRL + mouse-drag
    Box-select of themes 
  • CTRL + Alt + G
    Group selected themes (ancestor lvl.) 
  • CTRL + D
    Delete selected themes and related 
  • escape
    Closes side panel and groupping tool

Your data

Your data is totally anonymously stored in a database. Only you, by connecting with your Google account can help the system to know what is your data. The data is not encrypted (maybe in the future) so don't store too sensitive data in this tool, I'm not a security engineer :) (yet).
